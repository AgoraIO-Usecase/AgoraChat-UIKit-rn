# react-native-chat-library

## github

[repo](https://github.com/easemob/react-native-chat-library)

## install

1. `yarn` or `yarn install`
2. `yarn build`

## ios

1. `cd example/ios && pod install`
2. `cd .. && yarn run ios`

## android

`cd example && yarn run android`

## start with only server

`yarn start`
